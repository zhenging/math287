6.1 Definition of a Linear Transformation

Mapping
Let VVV and WWW be vector spaces. A mapping TTT from VVV into WWW is a rule that assigns to each vector vvv in VVV precisely on vector w=T(v)w=T(v)w=T(v) in WWW. We denote such a mapping by T:V→WT:V \to WT:V→W.
www is called the image of vvv.
vvv is called the preimage of www.

Examples
1. T: \R^3 \to \R^2, defined by T(x,y,z)=(x−y,x−2z)T(x, y, z) = (x-y, x-2z)T(x,y,z)=(x−y,x−2z)
2. T: \R^2 \to \R^2, defined by T(x,y)=(x+3,y)T(x, y) = (x+3, y)T(x,y)=(x+3,y)
3. T: \R \to \R, defined by T(x)=x+1T(x) = x+1T(x)=x+1

Linear Transformation
A mapping V→WV \to WV→W is called a linear transformation from V to W if it satisfies the following properties
(1) T(u+v)=T(u)+T(v)T(u+v) = T(u) + T(v)T(u+v)=T(u)+T(v) for all u,vu, vu,v in VVV.
(2) T(cv)=cT(v)T(cv) = cT(v)T(cv)=cT(v) for all vvv in VVV and all scalars ccc.
These properties are called the linearly properties.
VVV is called the domain of TTT.
WWW is called the codomain of TTT.

  • T(c1v1+c2v2)=c1T(v1)+c2T(v2)T(c_1v_1 + c_2v_2) = c_1T(v_1) + c_2T(v_2)T(c1​v1​+c2​v2​)=c1​T(v1​)+c2​T(v2​) for all v1,v2v_1, v_2v1​,v2​ in VVV and scalars c1,c2c_1, c_2c1​,c2​
  • T(0v)=0wT(0v) = 0wT(0v)=0w
  • T(−v)=−T(v)T(-v) = -T(v)T(−v)=−T(v) for all vvv in VVV.

Matrix of TTT
If T: \R^n \to \R^m is a linear transformation, then the matrix of TTT is A=[T(e1),T(e2),⋯T(en)]A = [T(e_1), T(e_2), \cdots T(e_n)]A=[T(e1​),T(e2​),⋯T(en​)].

  • Every m×nm\times nm×n matrix represents a linear transformation from \R^n to \R^m.
  • Eveyr linear transformation from \R^n to \R^m can be represented by an m×nm\times nm×n matrix.
