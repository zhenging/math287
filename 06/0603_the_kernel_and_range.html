6.3 The kernel and range of a linear transformation

Kernel
Let T:V→WT:V \to WT:V→W be a linear transformation. The set of all vectors vvv in VVV such that T(v)=0T(v) = 0T(v)=0 is called the kernel of the TTT and is denoted by Ker(T)\text{Ker}(T)Ker(T).

Range
The range of T:V→WT:V \to WT:V→W is the subset of WWW consisting of all transformed vectors from VVV. The range of T is denoted by Rng(T)\text{Rng}(T)Rng(T).

If T:V→WT:V \to WT:V→W is a linear transformation, then

  • Ker(T)\text{Ker}(T)Ker(T) is a subspace of VVV.
  • Rng(T)\text{Rng}(T)Rng(T) is a subspace of WWW.

If T:\R^n \to \R^m is a linear transformation, then

  • Ker(T)=nullspace(A)\text{Ker}(T) = \text{nullspace}(A)Ker(T)=nullspace(A), which is a subspace of \R^n
  • Rng(T)=colspace(A)\text{Rng}(T) = \text{colspace}(A)Rng(T)=colspace(A), which is a subspace of \R^m

General Rank-Nullity Theorem
If T:V→WT:V \to WT:V→W is a linear transformation and VVV is finite-dimensional, then

dim[Ker(T)]+dim[Rng(T)]=dim(V)\begin{aligned} \text{dim}[\text{Ker}(T)] + \text{dim}[\text{Rng}(T)] = \text{dim}(V) \end{aligned} dim[Ker(T)]+dim[Rng(T)]=dim(V)​

One-To-One / Onto
If T:V→WT:V \to WT:V→W is a linear transformation.

  • TTT is one-to-one iff the preimage of every www in the range consists of a single vector. That is whenever v1≠v2,T(v1)≠T(v2)v_1 \ne v_2, T(v_1) \ne T(v_2)v1​≠v2​,T(v1​)≠T(v2​).
  • TTT is onto iff every element of WWW has a preimage in VVV.
  • TTT is one-to-one iff Ker(T)=0\text{Ker}(T) = {0}Ker(T)=0.
  • TTT is onto iff Rng(T)=W\text{Rng}(T) = WRng(T)=W. (This is one is immediate from the definition.)

If T:V→WT:V \to WT:V→W is a linear transformation, and let VVV and WWW be finite-dimensional.

  • (1) TTT is one-to-one, then dim[V]≤dim[W]\text{dim}[V] \le \text{dim}[W]dim[V]≤dim[W].
  • (2) If TTT is onto, then dim[V]≥dim[W]\text{dim}[V] \ge \text{dim}[W]dim[V]≥dim[W].
  • (3) If TTT is both one-to-one and onto, then dim[V]=dim[W]\text{dim}[V] = \text{dim}[W]dim[V]=dim[W].
  • (4) If dim[V]=dim[W]\text{dim}[V] = \text{dim}[W]dim[V]=dim[W], then TTT is one-to-one iff TTT is onto.
