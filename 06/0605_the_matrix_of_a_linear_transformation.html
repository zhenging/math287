6.5 The Matrix of a Linear Transformation

Let VVV and WWW be vector spaces, with ordered bases B={v1,v2,⋯vn}B=\{v_1, v_2, \cdots v_n\}B={v1​,v2​,⋯vn​} and C={w1,w2,⋯wn}C=\{w_1, w_2, \cdots w_n\}C={w1​,w2​,⋯wn​}, respectively.

Let T:V→WT:V \to WT:V→W be a linear transformation
The m×nm\times nm×n matrix

[T]BC=[[T(v1)]c,[T(v2)]c,⋯[T(vn)]c,]\begin{aligned} [T]_B^C = \big[ [T(v_1)]_c, [T(v_2)]_c, \cdots [T(v_n)]_c,\big] \end{aligned} [T]BC​=[[T(v1​)]c​,[T(v2​)]c​,⋯[T(vn​)]c​,]​

is called the matrix representation of TTT relative to the base BBB and CCC.

If T:V=RnT:V = \R^nT:V=Rn and W=RmW = \R^mW=Rm are each equipped with the standard bases, then [T]BC[T]_B^C[T]BC​ is just the matrix of TTT defined earlier.

If V=WV=WV=W and T(v)=vT(v) = vT(v)=v for all vvv in VVV, then [T]BC[T]_B^C[T]BC​ is just the change of basis matrix from BBB to CCC defined earlier.

[T(v)]C=[T]BC[v]B[T(v)]_C = [T]_B^C[v]_B[T(v)]C​=[T]BC​[v]B​
