7.3 Diagonalization

Let AAA and BBB be n×nn\times nn×n matrices.

If AAA is triangular, then its eigenvalues are the entries on its main diagonal.

Similar

  • AAA is similar to BBB if there exist an invertible matrix SSS such that B=S−1ASB=S^{-1}ASB=S−1AS.
  • If AAA and BBB are similar, then they have the same eigenvalues (including multiplicity).

Proof if AAA is similar to BBB, then B=S−1ASB = S^{-1}ASB=S−1AS for some invertible matrix SSS. Thus

det(B−λI)=det(S−1AS−λI)=det(S−1AS−λS−1S)=det[S−1(A−λI)S]=det(S−1)det(A−λI)det(S)=det(A−λI)\begin{aligned} det(B-\lm I) &= det(S^{-1}AS -\lm I)\\ &= det(S^{-1}AS -\lm S^{-1} S)\\ &= det[S^{-1}(A -\lm I) S]\\ &= det(S^{-1})det(A -\lm I) det(S)\\ &= det(A -\lm I) \end{aligned} det(B−λI)​=det(S−1AS−λI)=det(S−1AS−λS−1S)=det[S−1(A−λI)S]=det(S−1)det(A−λI)det(S)=det(A−λI)​

where we have used the face that det(S−1)=1det(S)det(S^{-1}) = \dfrac{1}{det(S)}det(S−1)=det(S)1​ in the final step. Consequently AAA and BBB have the same characteristic polynomial and hence the same eigenvalues (and multiplicities).

Diagonalizable

  • If AAA is similar to a diagonal matrix, then AAA is diagonalizable.
  • A is diagonalizable iff AAA is nondefective.
