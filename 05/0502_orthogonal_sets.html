5.2 Orthogonal Sets of Vectors and Orthogonal Projections

Unit Vector
If a vector has norm of 1, then it is called a unit vector.

Normalization
If v is any nonzero vector in an inner product space V, then the vector u=v∣∣v∣∣u=\dfrac{v}{||v||}u=∣∣v∣∣v​ is a unit vector in the direction of vvv. The process of replacing vvv with uuu is called normalization.

Orthogonal
A set SSS of nonzero vectors in an inner product space VVV is called orthogonal if every pair of vectors in SSS is orthogonal. If, in addition each vector in the set is a unit vector, then S is called orthonormal.

Theorems

  • If SSS is an orthogonal set of nonzero vectors in an inner product space VVV, then SSS, is linearly independent.
  • If VVV is an inner product space of dimension nnn, then any orthogonal set of nnn vector is a basis of VVV.
