4.4 Spanning Sets

Let VVV be a vector space.
Let S={v1,v2,⋯v3}S = \{v_1, v_2, \cdots v_3 \}S={v1​,v2​,⋯v3​} be a subset of VVV.
Let c1,c2,⋯cnc_1, c_2, \cdots c_nc1​,c2​,⋯cn​ be scalars.

  • uuu is a linear combination of v1,v2,⋯vkv_1, v_2, \cdots v_kv1​,v2​,⋯vk​, iff u=c1v1+c2v2+⋯+ckvku= c_1v_1 + c_2v_2 + \cdots + c_kv_ku=c1​v1​+c2​v2​+⋯+ck​vk​
  • Forming all poosible linear combination of v1,v2,⋯vkv_1, v_2, \cdots v_kv1​,v2​,⋯vk​ generates a subset of VVV called the linear span of v1,v2,⋯vkv_1, v_2, \cdots v_kv1​,v2​,⋯vk​, denoted by Span{v1,v2,⋯vk}\text{Span}\{v_1, v_2, \cdots v_k\}Span{v1​,v2​,⋯vk​} or Span(S)\text{Span} (S)Span(S).
  • Span(S)\text{Span} (S)Span(S) is a subspace of VVV.
  • If Span(S)=T\text{Span} (S) = TSpan(S)=T, then SSS is a spanning set of TTT. We can say "SSS spans TTT" or "TTT is spanned by SSS".
  • Any 3 nonzero, noncoplanear vector in R3\R^3R3 span R3\R^3R3.
  • Any 2 nonzero, nonlinear vector in R2\R^2R2 span R2\R^2R2.
