4.6 Base and Dimension

Basis

A minimal spanning set of a vector space is called a basis of that vector space. The plural of basis is bases.

Let VVV be a vector space, and let S={v1,v2,v3,v4,v5}S = \{v_1, v_2, v_3, v_4, v_5\}S={v1​,v2​,v3​,v4​,v5​} be a basis of VVV
1. All bases of V have exactly 5 vectors.
2. The dimension of VVV is 5.
3. SSS has no "deadweight".
4. minimal spanning set

  • If you add any vectors to SSS, then the bigger set is still a spanning sef VVV.
  • If you take away vectors from SSS, then the samller set is not a spanninet of VVV.

5. maximumal independent set

  • If you add any vectors to SSS, then the bigger set is not an independenet.
  • If you take away vectors from SSS (provided you don't take the last one)hen t\he samller set is still an independent set.

6. Any set of 5 independent vectors is a basis.
7. Any set of 5 vectors that spans VVV is a basis.
8. Any spanning set of VVV must contain 5 vectors or more.
9. Any independent set must contain 5 vectors of less.
10. If TTT is a subspace of VVV, then dim(T)≤5\text{dim}(T) \le 5dim(T)≤5.
11. If TTT is a subspace of VVV, then any basis for TTT is part of a basis VVV.

Finite/Infinite-dimensional vector space

If a vector space VVV has a basis consisting of a finite number of vectors, then VVV is called a finite-dimensional vector space. Otherwise V is called an infinite-dimensional vector space.

If dim[V]=n\text{dim}[V] = ndim[V]=n, and SSS is a set of vectors, then the following statements are equivalent.

  • SSS is a basis.
  • SSS is linearly independent.
  • SSS spans VVV.

If VVV is a vector space with basis BBB, then every vector in VVV can be written uniquely as a linear combination of the vectors in BBB.
